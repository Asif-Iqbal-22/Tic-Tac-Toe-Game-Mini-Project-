# Tic-Tac-Toe Game in C

A simple console-based Tic-Tac-Toe game for two players.

## Features
- Two-player gameplay
- Display board
- Check for win/draw
- Play again option

## How to Run
1. Compile the program:

    gcc tictactoe.c -o ttt

2. Run:

    ./ttt
